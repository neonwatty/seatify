# TableCraft User Controls

Quick reference for all user interactions on desktop and mobile.

---

## Desktop Controls

### Canvas Navigation
| Action | Control |
|--------|---------|
| Pan canvas | Click and drag on empty area |
| Zoom in/out | Mouse wheel or +/- buttons |
| Zoom to fit | Double-click zoom percentage |

### Table Interactions
| Action | Control |
|--------|---------|
| Select table | Click on table |
| Multi-select | Cmd/Ctrl + click |
| Range select | Shift + click |
| Move table | Drag selected table |
| Nudge table | Arrow keys (10px) |
| Fine nudge | Shift + Arrow keys (1px) |
| Delete table | Delete/Backspace key |
| Duplicate | Cmd/Ctrl + D |
| Deselect | Escape or click empty area |

### Guest Interactions
| Action | Control |
|--------|---------|
| Assign to table | Drag guest chip to table |
| Unassign | Drag guest off table to canvas |
| Edit guest | Double-click guest chip |
| Context menu | Right-click guest |

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + F` | Search canvas |
| `Cmd/Ctrl + D` | Duplicate selected |
| `Delete` / `Backspace` | Delete selected |
| `Escape` | Deselect / Close panel |
| `?` | Show keyboard shortcuts help |
| `Arrow keys` | Nudge selected (10px) |
| `Shift + Arrow` | Fine nudge (1px) |

### Context Menus (Right-click)
- **On table**: Edit, Duplicate, Delete, Select all guests
- **On guest**: Edit, Remove from table, Delete
- **On canvas**: Add table, Add guest, Paste

## Mobile / Touch Controls

### Canvas Navigation
| Action | Control |
|--------|---------|
| Pan canvas | Two-finger drag, or single-finger on empty area |
| Zoom | Pinch gesture |

### Table Interactions
| Action | Control |
|--------|---------|
| Select table | Tap on table |
| Move table | Long-press (200ms) then drag |
| Open properties | Tap selected table |

### Table Properties Panel (Mobile)
The panel slides up from the bottom when a table is selected.

| Control | Action |
|---------|--------|
| Drag handle | Swipe down to dismiss |
| Backdrop | Tap to dismiss |
| Close (×) | Tap to dismiss |
| Prev/Next arrows | Navigate between tables |
| Duplicate button | Create copy of table |
| Rotate button | Rotate table 45° |
| Delete button | Delete table (with confirmation) |
| Shape buttons | Change table shape |
| Capacity +/- | Adjust seating capacity |
| Advanced Settings | Expand for size presets & dimensions |

### Guest Interactions
| Action | Control |
|--------|---------|
| Assign to table | Long-press (200ms) then drag to table |
| Edit guest | Double-tap guest chip |

### Sidebar (Mobile)
| Control | Action |
|---------|--------|
| Guests button | Open sidebar drawer |
| Backdrop | Tap to close sidebar |
| Close button | Close sidebar |

## Grid & Alignment Tools

| Control | Function |
|---------|----------|
| Grid toggle | Show/hide grid overlay |
| Snap toggle | Enable/disable snap-to-grid |
| Guides toggle | Enable/disable alignment guides |
| Grid size | Choose 20px, 40px, or 80px grid |

## View Modes

| View | Description |
|------|-------------|
| Canvas | Main floor plan editor |
| Guest List | Table view of all guests |
| Relationships | Visual relationship matrix |

## Touch Target Sizes

All interactive elements meet the 44px minimum touch target requirement for accessibility on touch devices.
